# Overview

As a cybersecurity specialist, I wanted to expand my skill set by learning JavaScript and understanding how web applications work at a fundamental level. 
Having a strong grasp of web technologies is crucial for identifying vulnerabilities, conducting security assessments, and understanding attack vectors in modern web applications. 
This portfolio website serves as both a learning project to master JavaScript fundamentals and a professional platform to showcase my cybersecurity projects and skills.I created a fully responsive personal portfolio website using vanilla JavaScript, HTML5, and CSS3. 
The site features smooth scrolling navigation, animated background effects using the Canvas API, intersection observer for scroll-triggered animations, and dynamic DOM manipulation. By building this from scratch without frameworks, I gained hands-on experience with core JavaScript concepts including ES6 syntax, event handling, object-oriented programming, and browser APIsâ€”all of which are essential knowledge for understanding client-side security risks.

My purpose for writing this software was to deepen my understanding of JavaScript's capabilities and how it interacts with web browsers. As a cybersecurity professional, understanding how JavaScript manipulates the DOM, handles user input, and manages client-side logic helps me better assess web application security, identify potential XSS vulnerabilities, and understand the attack surface of modern web applications. This project demonstrates practical application of JavaScript while building a tool I can use professionally.
